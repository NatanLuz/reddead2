# Red Dead Redemption 2 - Website Responsivo

Aplicação web estática responsiva desenvolvida com HTML, CSS e JavaScript, inspirada no jogo _Red Dead Redemption 2_, com foco em experiência visual, navegação fluida e adaptação para múltiplos dispositivos.

## 🎯 Proposta de Valor

Construir uma interface web responsiva com foco em experiência do usuário, organização visual e uso de animações para navegação fluida.

**Benefícios principais:**

- Experiência consistente entre diferentes dispositivos
- Navegação fluida entre seções
- Interface clara e bem estruturada
- Uso de animações para melhoria de UX

---

## ⚙️ Funcionalidades

- Layout responsivo (desktop, tablet e mobile)
- Menu de navegação fixo com rolagem suave
- Menu hambúrguer animado para dispositivos móveis
- Seções estruturadas (História, Personagens, Mundo, Missões e Trilha Sonora)
- Animações ao rolar a página (AOS)
- Efeitos de hover e transições suaves
- Integração com conteúdo externo (ex.: Spotify)

---

## 🏗️ Arquitetura / Estrutura

Separação de responsabilidades entre estrutura, apresentação e comportamento:

- **HTML** → marcação semântica do conteúdo
- **CSS** → layout, responsividade e animações
- **JavaScript** → interações e controle de comportamento da interface

Estrutura de arquivos:

```bash
reddead2-project/
├── index.html
├── style.css
├── animacao.js
├── assets/
│   └── imagens e ícones
```

## 🚀 Instalação

### Pré-requisitos

- Navegador moderno (Chrome, Edge, Firefox, etc.)

### Passos

1. Clonar o repositório
2. Acessar a pasta do projeto
3. Abrir o arquivo principal

```bash
git clone https://github.com/archivesysl/reddead2-project.git
cd reddead2-project
```

4. Abrir `index.html` no navegador

Opcional:

- Utilizar **Live Server (VS Code)** para ambiente de desenvolvimento
