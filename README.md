## Sobre o Projeto

Este projeto é um website totalmente responsivo dedicado ao jogo **Red Dead Redemption 2**, desenvolvido com foco em oferecer uma experiência consistente em todas as plataformas. Também desenvolvido para aprender e aperfeiçoar em conhecimentos sobre html, css e javascript.

O site apresenta elementos-chave do jogo, como:

- História
- Personagens
- Mundo aberto
- Missões
- Trilha sonora

Tudo foi pensado para funcionar bem em qualquer dispositivo (desktop, tablet e mobile).

---

## Tecnologias Utilizadas

- **HTML5** – Estrutura semântica do conteúdo
- **CSS3** – Estilização avançada, layout responsivo e animações
- **JavaScript** – Interatividade, comportamentos dinâmicos e animações
- **Font Awesome** – Ícones e elementos visuais
- **AOS (Animate On Scroll)** – Animações ao rolar a página
- **Vercel** – Hospedagem e deploy do projeto

## Funcionalidades

### 1. Design Responsivo

- Layout adaptável para **desktop**, **tablet** e **mobile**
- Menu hambúrguer em dispositivos móveis
- Imagens e conteúdos otimizados para diferentes tamanhos de tela
- Sistema de grid flexível

### 2. Navegação

- Menu fixo e transparente
- Rolagem suave entre as seções
- Menu hambúrguer animado no mobile
- Acesso rápido a todas as seções da página

### 3. Seções Principais

- **História**: Linha do tempo interativa com os capítulos do jogo
- **Personagens**: Cards responsivos com informações dos personagens
- **Mundo Aberto**: Destaque para as principais localidades
- **Missões**: Descrição detalhada de atividades e missões
- **Trilha Sonora**: Integração com Spotify

### 4. Elementos Visuais

- Animações suaves de entrada
- Efeitos de hover interativos
- Transições fluidas
- Ícones temáticos
- Símbolo de honra personalizado

## Últimas Atualizações

### Melhorias de Responsividade

**Menu Mobile**

- Implementação de menu deslizante
- Animações suaves de abrir/fechar
- Uso de _backdrop filter_ para melhor legibilidade
- Fechamento automático após seleção de item

**Linha do Tempo (Timeline)**

- Reorganização para visualização vertical em telas menores
- Ajustes de espaçamento e alinhamento
- Otimização das animações de entrada
- Melhor legibilidade em aparelhos pequenos

**Layout Adaptativo**

- Otimização de _breakpoints_ (480px, 768px, 1024px)
- Ajustes de tipografia para cada dispositivo
- Espaçamentos mais consistentes
- Sistema de grid aprimorado

**Performance**

- Otimização de imagens
- _Lazy loading_ de recursos quando possível
- Animações mais leves e eficientes
- Transições mais suaves

### Breakpoints Utilizados

- **Mobile:** < 480px
- **Tablet:** 480px – 768px
- **Desktop Pequeno:** 768px – 1024px
- **Desktop Grande:** > 1024px

## Como Executar o Projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/archivesysl/reddead2-project.git
```

### 2. Abrir no navegador

- Abra o arquivo `index.html` diretamente no navegador
  ou
- Sirva o projeto com uma extensão como **Live Server** (VS Code) para uma melhor experiência de desenvolvimento

### 3. Desenvolvimento

- Estilos: editar o arquivo `style.css`
- Animações: ajustar o arquivo `animacao.js`
- Conteúdo: atualizar o arquivo `index.html`
